# デジタル度ダッシュボード

デジタル度ダッシュボードは、日本の都道府県のデジタル化の程度を可視化するウェブアプリケーションです。

## 機能
- 都道府県のデジタル度指標（DCI、ネット利用、デジタル公共サービス、コネクティビティ、人的資本）の値とランキングを表示
- 都道府県をクリックすると詳細情報を確認できる

## データ
本アプリケーションでは、[野村総合研究所(NRI)による「DCIにみる都道府県別デジタル度」のCSVデータ](https://github.com/code4fukui/dci-stat/)を使用しています。
